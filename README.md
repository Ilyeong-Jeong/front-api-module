# front-api-module 
**Front-end에서 REST API, GQL을 사용하기 위한 API 모듈**

[API Module](https://github.com/Ilyeong-Jeong/front-api-module/tree/main/src/modules) api.ts 확인.

## Tech Stack
Node v14 base
- React v17
- TypeScript v4
- Webpack v5
- React Router v5

## Usage
**Install dependencies**
```
npm install
```

**Run development mode (only)**
```
npm run dev
```
